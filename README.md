# 🖊️ КанцТовары - Интернет-магазин канцелярских товаров

## 📖 Описание

**КанцТовары** — это десктопное приложение для управления интернет-магазином канцелярских товаров. Разработано в рамках практического экзамена по дисциплине «Осуществление интеграции программных модулей».

Приложение позволяет:
- Просматривать каталог товаров с поиском и фильтрацией
- Управлять корзиной покупок
- Оформлять заказы с валидацией данных
- Управлять товарами и заказами через административную панель

## 🛠️ Стек технологий

- **Python 3.10+** — язык программирования
- **PyQt5** — графический интерфейс
- **SQLite3** — хранение заказов
- **JSON** — хранение товаров и корзины
- **logging** — логирование

## 🚀 Установка и запуск

### Способ 1: Автоматическая установка (Windows)

1. **Клонируйте репозиторий (см. шаг 1 выше)**
2. **Запустите скрипт установки:**
    setup.bat - Этот скрипт автоматически создаст виртуальное окружение и установит зависимости.
3. Запустите приложение:
    run.bat

### Способ 2: Ручная установка

1. **Клонируйте репозиторий:**
   git clone https://github.com/SkkroLL/shop-candidate
   cd shop-candidate
2. **Создайте и активируйте виртуальное окружение:**
    python -m venv .venv
    **Windows:**
    .venv\Scripts\activate
    **macOS/Linux:**    
    source .venv/bin/activate
3. **Установите зависимости:**
    pip install -r requirements.txt
4. **Установите зависимости:**
    python src/main.py
